# AWS-cloud-security-labs-
# 🔐 Secure EC2 Access with Session Manager (No SSH)

## 📌 Overview
This project demonstrates how I improved the security of an EC2 instance by replacing SSH access with AWS Systems Manager Session Manager.

## 🚀 Key Highlights
- Removed SSH (port 22) exposure
- Implemented IAM-based access control
- Enabled secure, auditable access
- Troubleshot real-world AWS networking issues

## 🧰 Services Used
- Amazon EC2
- AWS Systems Manager (SSM)
- AWS IAM
- Amazon VPC

Successfully eliminated public access to the EC2 instance while maintaining secure administrative access using SSM.
